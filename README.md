# 💰 Expense Tracker

A simple **offline-first desktop expense tracker** built with **Electron**, **Node.js**, and **React.js**.

🎥 **Demo Video:** https://youtu.be/988Q8hHytX4

ExpenseTracker is a project I built to solve a personal need: keeping all my receipts, bills, and recurring invoices organized in one place. The goal is to make financial tracking easier and provide a quick overview of what’s already paid and what’s still pending each month.

---

## ✨ Features

- 📌 Store recurring invoices  
- 📊 View total monthly expenses  
- 🧾 List all saved receipts and bills  
- ⏰ Track pending payments  
- 💾 Fully offline-first using a local SQLite database  
- 🖥️ Desktop application for Windows (Electron)

---

## 🛠 Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Sequelize  
- **Database:** SQLite  
- **Desktop Framework:** Electron  

---

## 📦 Windows Installer

A ready-to-use Windows installer is available inside:

```bash
builds/
```

---

## 🚀 Building the App Manually

To install dependencies and generate the desktop build:

```bash
npm install
npm run electron:dist
```
